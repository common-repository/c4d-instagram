== Description ==

This plugin allow you display your instagram on your wordpress site 

Just insert below shortcode into widget or post or anywhere: 

[c4d-instagram username="your-instagram-username"]

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

1. Upload `c4d-instagram` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
